Python 2 的历史

  • 2.0 发布于 2000 年
  • 2.2/2.3 2001年/2004年
  • PEP 3000 发布于 2006 年
  • 2.7 发布于2010年
  • 2020 年 Python 2 结束维护周期

Python 2 的缺陷

  • 编码的支持
  • 语义的分歧
  • 内建系统设计不合理

Python 2 的缺陷之: 编码问题


a = "abcd"

好了，这段代码代表着什么含义

  • A. 包含了”a”, “b”, “c”, 和 “d” 的字符串
  • B: 表示 97, 98, 99, 和 100 的字节数据

答案是，以上都可能
